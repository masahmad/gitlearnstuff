INSTALLATION:
	1.	Upload the files to the server
	2.	Create apps or projects for Twitter, Facebook and Google. Search the internet for a tutorial. 
		You have to set the allowed callback/redirect urls on the facebook, twitter and google site. You can get the callback/redirect url with the page what_callback_url_do_i_have_to_use.php (just open it in your browser)
	3.	Set the client_id or customer_keys and secrets in the config.php file. 
	4.	The script now works, edit the end of callback.php to your needs.
